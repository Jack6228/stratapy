Installation
============

stratapy runs in Python, but there are few ways this can be achieved. 

- If you wish to **install Python on your machine** for the first time, we recommend following the steps in the `First Time Setup <first_time_setup.html>`__ guide below.

- Alternatively, you can **use an online platform** such as Google Colab to run Python and stratapy **without installing anything** on your local machine.

- If you **already have Python installed** and are familiar with using it, you can skip to `Installation <installation.html>`__ to install stratapy directly.

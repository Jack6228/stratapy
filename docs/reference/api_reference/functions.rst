Primary Functions
--------------------

These functions form the core functionality of the stratapy package and are categorised into four main areas: loading data, plotting functions (beyond ``Log.plot()`` which is documented in the :doc:`Log Class <LogClass>` section), and customising lithologies, features and minerals.

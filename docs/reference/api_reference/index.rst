API Reference
=============

This section contains full documentation for all of the functions and classes available in stratapy. Primary Functions highlights the main functions available in the core module, such as loading data and standalone plotting functions. The Log Class section details the methods available for the LogObject class which is created following a call to ``sp.load()``, and the MultiLogFigure Class section details the methods available for the MultiLogFigure class which is created following calls to ``sp.correlated_logs()`` or ``sp.multi_fig()``.
